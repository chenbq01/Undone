  • 如何下载沙发院线客户端？

    方法1：登陆沙发院线网站(www.sofamovie.cn)，根据手机系统选择对应的客户端下载安装。

    方法2：手机登陆各大应用市场下载并安装 “沙发院线” 。如Appstore，豌豆荚等。

    方法3：在微信中查找订阅账号，订阅“沙发院线“订阅号。订阅成功后即可按提示获取下载链接。

    友情提示：

    1.建议用户在WIFI环境下载沙发院线手机客户端。

    2.目前沙发院线手机客户端支持iOS和Android设备。

  • 如何注册成为沙发院线用户？

    1.首次打开沙发院线手机客户端，选择底栏菜单最右侧“我的”，进入个人中心页面。

    2.选择右上角“注册”，进入注册信息页。

    3.输入个人信息注册：

    3.1 每个手机号码仅限注册一次，不可重复注册；

    3.2 为保护用户的账户安全，密码为8-16为数字与字母结合；

    3.3 机顶盒卡卡号可直接扫描读取，如无法扫描读取时可直接输入，同一机顶盒卡卡号支持被多个手机号绑定；

    3.4 选择机顶盒卡所在城市。如卡号与所在城市不符，将无法完成注册；

    3.5 本页所有注册项均为必填项。

    4.输入完成，注册成功。

    友情提示：用户在插拔用户卡时，请关闭机顶盒电源，以免出现机顶盒硬件故障。

  • 点播流程

    1.打开沙发院线手机客户端；

    2.点击“本周主打”或“本周热播”选择影片；

    3.点击“点播到电视”，选择日期、场次、客厅；

    4.完成时间、场次、客厅后，可加入购物车，或直接支付。

    友情提示：开场前10分钟内的影片不再提供点播。

  • 支付流程

    1.完成订单后页面将跳转至支付页面；

    2.支付成功后出现点播成功提示页，注册手机号将收到点播成功短信提示。

    友情提示：

    1.沙发院线手机客户端目前仅支持支付宝支付；

    2.银行卡支付（含境内储蓄卡、信用卡）将于近期开通。

  • 订单查询

    1.已支付成功的所有订单可在“订单”-“已成功”订单中查询；

    2.支付未成功的所有订单可在“订单”-“待支付”订单中查询，并可重新支付。

  • 观看频道

    如已成功点播并支付，可以从以下方式查找播放频道名称：

    1.登录沙发院线客户端，点击底栏“订单”菜单；

    2.在“已成功”订单中查看点播记录及播放频号及频道名称等信息。

  • 为什么手机号无法注册？

    沙发院线客户端账户支持中国移动、联通、电信等通信运营商的11位手机号进行注册。

    您的手机号无法注册可能出现以下几种情况：

    1.手机号位数不对，或非真实手机号码；

    2.用户的手机号已经注册过账号，用户可通过“忘记密码”功能来找回密码。

    注：如该手机号未注册过沙发院线客户端账户，将无法接受短信验证码。

  • 为什么输入机顶盒用户卡数字串后无法注册成功？

    如用户的机顶盒用户卡在注册时无法成功，可能有以下几种情况：

    1.所输入的机顶盒用户卡卡号位数不对，或非真实卡号；

    2.所输入的机顶盒用户卡卡号与用户所选择的地区不一致，系统会自动验证卡号与地区，如验证不通过则无法注册成功。

  • 如何修改或删除已经绑定的用户卡及备注信息？

    对已经绑定的机顶盒用户卡，可通过以下方式进行编辑：

    1.登录个人中心；

    2.长按已绑定的机顶盒用户卡条目，出现编辑选项；

    3.根据需要选择“修改智能卡信息”，可编辑机顶盒用户卡卡号及备注；或“删除智能卡”。

  • 如何添加新的机顶盒卡号？

    如需要在已注册账户下添加新的机顶盒卡号，请按以下步骤操作。

    1.登录个人中心，并点击“添加智能卡”；

    2.扫描或输入机顶盒用户卡，选择正确的对应城市，填入备注信息，即可绑定成功。

  • 是否可以添加异地的机顶盒卡号？

    对沙发院线业务所覆盖的地区，均可绑定当地机顶盒用户卡。如所需绑定的地区暂未覆盖沙发院线业务，则绑定不成功。沙发院线目前覆盖地区，可通过登录手机客户端后，点击左上角“地区”了解。

  • 如何变更注册手机号？

    注册沙发院线手机客户端账户时所用的手机号码，即为用户的账户名称，无法变更。如您使用了新的手机号码，建议重新注册。

  • 同一账户可以绑定几张机顶盒用户卡？

    每个手机号码即为一个沙发院线手机客户端账户，可以最多绑定5张机顶盒用户卡。所绑定机顶盒用户卡地区不限（须沙发院线所覆盖地区）。

  • 一张机顶盒用户卡可以授权几个注册账号？

    每张机顶盒用户卡所授权的注册账号暂时不限。

    1.多个沙发院线手机客户端账户可以同时绑定同一张机顶盒用户卡。

    2.不同账户可为同一张机顶盒用户卡进行点播。

  • 如何找回注册账号密码？

    如忘记沙发院线手机客户端登录密码，可按如下方式操作：

    1.打开沙发院线手机客户端，点击底栏“我的”菜单，进入个人中心页面；

    2.点击“忘记密码”，进入“重置密码”页面后进行密码重置。

  • 如何变更账户密码？

    如需要变更沙发院线手机客户端登录密码，可按如下方式操作：

    1.登录沙发院线手机客户端，进入个人中心页面；

    2.点击“修改密码”，输入原密码及新密码后，即可使用新密码重新登录。

  • 为什么无法绑定新的机顶盒用户卡？

    如在添加新的的机顶盒用户卡时不成功，可能有以下几种情况：

    1.所输入的机顶盒用户卡卡号位数不对，或非真实卡号；

    2.所输入的机顶盒用户卡卡号与所选择的地区不一致，系统会自动验证卡号与地区，如验证不通过则无法添加成功。

  • 已经点播并支付成功，在哪个播放频道？

    如已成功点播并支付，可以从以下方式查找播放频道名称：

    1.登录沙发院线客户端，点击底栏“订单”菜单；

    2.根据用户的订单查看播放频道名称。

  • 已经点播并支付成功，为什么电视没有播放电影？

    如已成功点播并支付，到播放时间时未播放影片可能有以下几种情况：

    1.可能打开了错误的频道；

    2.可能点播订单未支付成功，可到支付成功订单中查看；

    3.沙发院线与世界标准时间一致，可能与用户的时钟时间略有差异，一般差异时间不超过5分钟；

    4.如已确定订单支付成功，且在点播时间10分钟后打开了正确的频道仍未播放电影，请客服拨打本地有线电视客服热线。

  • 没有支付宝账户，如何支付？

    支付宝账户目前已支持无账户支付，用户可直接输入银行卡号进行支付。沙发院线将于近期开通银行卡直接支付功能（含境内储蓄卡、信用卡）。

  • 为什么在待付款或购物车里的订单无法支付？

    在待付款或购物车里的订单无法支付时，可能遇到以下问题：

    1.所支付订单内的影片场次距离开播时间在10分钟内。系统不支持10分钟内场次的点播、支付；

    2.极少数情况下，用户所支付订单内的影片场次已被下片；

    3.如非以上两种情况，建议用户重新登录客户端后另行点播支付。

  • 为什么无法即点即看？

    为了用户的观影准备和体验，沙发院线对10分钟以内的影片场次不提供点播。同一影片可选择下一场次或观看其他影片。

  • 已经开播的影片是否可以回放？

    沙发院线为用户提供影院式观影服务，用户如误点或误场观影，无法提供回放服务。

  • 已点播的影片可以暂停或延时播放吗？

    沙发院线为用户提供影院式观影服务，观影中途无法暂停或延时观看已点播影片，不过您可以在家尽情享受美食或亲密。
